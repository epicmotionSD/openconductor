## Installation

All new servers can be discovered and installed via the OpenConductor CLI:

```bash
# Search for servers
openconductor discover "stripe"

# Install a server
openconductor install stripe-mcp

# List all installed servers
openconductor list
```

Or browse them at [openconductor.ai/discover](https://openconductor.ai/discover)

## Technical Details

### Adding Process

All servers were:
1. Researched from official MCP repositories
2. Verified for quality and maintenance
3. Added with complete metadata (description, tags, repository info)
4. Tested via the OpenConductor API
5. Made immediately available via CLI and web interface

### Data Sources

Servers were sourced from:
- [modelcontextprotocol/servers](https://github.com/modelcontextprotocol/servers) - Official MCP servers
- [wong2/awesome-mcp-servers](https://github.com/wong2/awesome-mcp-servers) - Community list
- [habitoai/awesome-mcp-servers](https://github.com/habitoai/awesome-mcp-servers) - Curated list

### Database Schema

Each server includes:
- Basic info (slug, name, tagline, description)
- Repository details (URL, owner, name)
- NPM package information
- Category and tags
- Verification status
- Installation stats
- Version history
