**Desktop Cleanup**:
```bash

# Disable notifications
# System Preferences ‚Üí Notifications ‚Üí Turn on Do Not Disturb
```

**Claude Desktop**:
- [ ] Clear conversation history (start fresh)
- [ ] Use dark theme (Settings ‚Üí Appearance)
- [ ] Maximize window (full screen)
- [ ] Close all other apps

---

### Step 2: Install Recording Software (10 minutes)

**macOS - QuickTime (Easiest)**:
1. Already installed ‚úì
2. Open QuickTime Player
3. File ‚Üí New Screen Recording
4. Click Options:
   - Microphone: Select your mic
   - Quality: Maximum
   - Show Mouse Clicks: Off (we'll add in post)
5. Ready to record

**macOS/Windows/Linux - OBS Studio (Professional)**:
```bash
# macOS
brew install --cask obs

# Ubuntu/Debian
sudo apt install obs-studio

# Windows
# Download from obsproject.com
```

**OBS Setup**:
1. Open OBS Studio
2. Create New Scene: "Claude Demo"
3. Add Source ‚Üí Display Capture (macOS) or Window Capture (Windows/Linux)
4. Settings ‚Üí Video:
   - Base Resolution: 1920x1080
   - Output Resolution: 1920x1080
   - FPS: 30
5. Settings ‚Üí Output:
   - Recording Format: mp4
   - Encoder: x264
   - Video Bitrate: 10000 Kbps
6. Settings ‚Üí Audio:
   - Desktop Audio: Disabled (we want silence during screen recording)
   - Mic/Auxiliary: Your microphone
7. Click "Start Recording" to test (record 10 seconds, verify quality)

---

### Step 3: Verify OpenConductor Stack Install Works (5 minutes)

**Critical: Test the actual install before recording!**

```bash
# Ensure CLI is latest version
npm install -g @openconductor/cli

# Verify version
openconductor --version
# Should show: 1.2.0 or higher

# Test stack list command
openconductor stack list
# Should show beautiful table with 3 stacks

# DO NOT install a stack yet (save for recording)
# But verify the command works
openconductor stack show coder
# Should display coder stack details
```

**If anything fails, fix it now before recording!**

### Voiceover Setup

**Audacity Setup** (FREE):
```bash
# Install Audacity
# macOS
brew install --cask audacity

# Ubuntu
sudo apt install audacity

# Windows
# Download from audacityteam.org
```

**Recording Settings**:
1. Open Audacity
2. Select your microphone (dropdown at top)
3. Sample Rate: 44100 Hz
4. Format: 32-bit float
5. Click red record button

**Room Prep**:
- [ ] Close windows (street noise)
- [ ] Turn off fans, AC, refrigerator (if possible)
- [ ] Put phone in airplane mode
- [ ] Hang "Recording - Do Not Disturb" sign
- [ ] Use blankets/pillows to dampen echo (or record in closet)

**Mic Technique**:
- [ ] Position mic 6-8 inches from mouth
- [ ] Slightly off-axis (not directly in front, 45¬∞ angle)
- [ ] Use pop filter (or improvise with pantyhose over wire hanger)
- [ ] Stand up (better breath control) or sit with straight posture
- [ ] Drink water before starting

---

### Recording the Voiceover

**Script**: [V1.2.0_PRODUCTION_SCRIPTS.md](V1.2.0_PRODUCTION_SCRIPTS.md) - VIDEO 1

**Process**:
1. **Slate each section**: "Hero Video, Scene 1, Take 1"
2. **Record section**: Read naturally, with energy
3. **Pause 3 seconds**: Silence between sections (easier to edit)
4. **Repeat**: "Hero Video, Scene 1, Take 2"
5. **Move to next section**

**Sections to record** (in order):
```
‚ñ° Scene 1: Cold Open (0:00-0:10)
‚ñ° Scene 2: Problem Setup (0:12-0:18)
‚ñ° Scene 3: Problem Details (0:18-0:30)
‚ñ° Scene 4: Transition (0:30-0:33)
‚ñ° Scene 5: Stacks Introduction (0:33-0:50)
‚ñ° Scene 6: Install Demo (0:50-1:15)
‚ñ° Scene 7: System Prompt (1:15-1:30)
‚ñ° Scene 8: Real Usage (1:30-1:50)
‚ñ° Scene 9: All Stacks (1:50-2:15)
‚ñ° Scene 10: CTA (2:15-2:30)
```

**Tips**:
- **Energy**: Enthusiastic but not over-the-top
- **Pacing**: Speak slightly slower than normal conversation
- **Emphasis**: Stress key words ("10 seconds", "zero configuration", "instant")
- **Breath**: Take breaths between sentences (edit out later)
- **Mistakes**: Don't stop if you mess up, just pause 2 seconds and re-read the sentence

**Record 2-3 takes per section** (pick best in editing)

### Post-Recording Cleanup

**In Audacity**:
1. **Noise Reduction**:
   - Select 1 second of silence (just background noise)
   - Effect ‚Üí Noise Reduction ‚Üí Get Noise Profile
   - Select All (Cmd+A)
   - Effect ‚Üí Noise Reduction ‚Üí OK (default settings)

2. **Normalize**:
   - Select All
   - Effect ‚Üí Normalize ‚Üí OK (-1.0 dB)

3. **Export**:
   - File ‚Üí Export ‚Üí Export as WAV
   - Save as: `hero-video-voiceover.wav`

**Duration**: 10 minutes for cleanup

## üÜò Troubleshooting

### "Stack install fails during recording"
**Solution**: Test install beforehand, use `openconductor stack uninstall coder` between takes

### "Recording is laggy/choppy"
**Solution**: Close all other apps, lower OBS bitrate to 5000 Kbps, use QuickTime instead

### "I messed up the voiceover"
**Solution**: Just pause 2 seconds, re-read the sentence, keep recording

### "Claude's response is different than expected"
**Solution**: Record multiple takes, or guide the response with more specific query

### "System prompt didn't auto-copy to clipboard"
**Solution**: Manually copy from terminal output, paste in Claude (still looks good on camera)

### "I don't have a PostgreSQL database for the demo"
**Solution**: Use this alternative query instead:
```
"Search for Next.js 14 best practices, save the findings to memory, and create a checklist I can reference later."
```
This uses Brave Search + OpenMemory (no database needed)
